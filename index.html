Array Modified Methods

Push

Adds one or more elements at the end of an array.

Pop

Removes last lement from an array.

Shift

Removes first element from an array.

UnShift

Adding one or more elements at starting of an array.

Splice

Adds/Removes an element from an particular index.

Sort

Ascending/Descending of an array
For strings it will shown default but For numbers it will shown incorrect to achieve this we use compare function.

Reverse

It's reverse the order of an array

Fill

Filling a specified value with in specific indexes

CopyWithIn

Copying some part of an array and placed in the targeted position

Array Don't Modified Methods

Concat

Merging of an two arrays

Slice

Extracting a part from an existing array and returns new array

Includes

If an array contains specific element or not

IndexOf

Returns the first index of an given element

LastIndexOf

Returns the last index of an given element

Join

Joining of array elements and returns as string

ToString

Converts array in to a string seperated by commas

ToLocalString

Converts array in to a string using local-specific format

Flat

If there is any nested array as deep it will remove nested array as mentioned value

FlatMap

It is the combination of both flat and map method
