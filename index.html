Mutator Methods (Array Modified Methods)

Push

Adds one or more elements at the end of an array.

Pop

Removes last lement from an array.

Shift

Removes first element from an array.

UnShift

Adding one or more elements at starting of an array.

Splice

Adds/Removes an element from an particular index.

Sort

Ascending/Descending of an array
For strings it will shown default but For numbers it will shown incorrect to achieve this we use compare function.

Reverse

It's reverse the order of an array.

Fill

Filling a specified value with in specific indexes.

CopyWithIn

Copying some part of an array and placed in the targeted position.

Accessor Methods (Array Don't Modified Methods)

Concat

Merging of an two arrays.

Slice

Extracting a part from an existing array and returns new array.

Includes

If an array contains specific element or not.

IndexOf

Returns the first index of an given element.

LastIndexOf

Returns the last index of an given element.

Join

Joining of array elements and returns as string.

ToString

Converts array in to a string seperated by commas.

ToLocalString

Converts array in to a string using local-specific format.

Flat

If there is any nested array as deep it will remove nested array as mentioned value.

FlatMap

It is the combination of both flat and map method.

Iteration Methods (Looping Methods)

ForEach

Iterating over an array without returning any value.

Map

Iterating over an array and returns a new array with transformed values.

Filter

Creating a new array by filtering.

Reduce

Having accumulator property which can store the previous value.

ReduceRight

Same as reduce method but it will process from right to left.

Some

Returns true or false atleast one element passed the condition.

Every

Return true or false based on the condition if every element passed the test.

Find

Returns first element value in an array if the condition has passed.

FindIndex

Returns first element index value in an array if the condition has passed.

Keys

Which returns new array iterator (return indexes).

Values

Which returns new array iterator (return values).

Entries

Which returns both keys and values.

Utility Methods

Array.isArray

Returns true/false whether the passed variable is array or not.

Array.from

Which converts in to an array

Array.of

which creates a new array.
